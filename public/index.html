PULSE HTTP

Creative HTTP Server

A custom, high-performance HTTP/1.1 network engine & MVC framework built from scratch.

PS#1 Web Server Core

Under The Hood

Pulse uses Node's low-level net.Socket duplex streams to read, buffer, parse, and respond to incoming raw HTTP/1.1 TCP data streams. By checking Content-Length limits dynamically, Pulse fully mitigates packet fragmentation bugs common in raw TCP servers.

Framework Capabilities:

  • TCP Socket Accumulator - Intelligently buffers chunked HTTP requests, parsing headers and waiting for complete body payload delivery.
  • Regex Route Router - Full routing support with params matching (e.g. /api/users/:id) and HTTP verbs.
  • Express-Style Middleware Piping - Executes serial middleware chains with full (req, res, next) signature control.
  • Pulse Template Engine - Custom templating engine supporting list loops (#each), conditionals (#if), and value bindings.
  • Declarative Validator - Built-in type coercion and validation middleware.
